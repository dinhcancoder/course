Khóa học HTML, CSS cơ bản đến nâng cao

Bài 1: Cấu trúc file HTML.

Phần DOCTYPE: khai báo kiểu văn bản. Nó cho trình duyệt biết bạn đang sử dụng phiên bản HTML nào.
Đây là phần rất quan trọng nhưng lại thường bị bỏ qua trong nhiều tài liệu về nhập môn HTML. Có rất nhiều loại tài liệu khác nhau trên web, và một trình duyệt chỉ có thể hiển thị một tài liệu HTML chính xác 100% nếu nó biết phiên bản HTML nào mà tài liệu đó đang sử dụng.
Phần khai báo DOCTYPE ở hình minh họa là của phiên bản HTML5. Phần tiêu đề: bắt đầu bằng thẻ và kết thúc bởi thẻ. Phần này chứa tiêu đề mà được hiển thị trên thanh điều hướng của trang web. Tiêu đề là phần nội dung nằm giữa cặp thẻ và.
Ngoài ra nó còn chứa một số khai báo khác, các thông tin phục vụ SEO mà chúng ta không đề cập ở đây. Phần thân: nằm sau phần tiêu đề. Nó bao gồm văn bản, hình ảnh và các liên kết mà bạn muốn hiển thị trên trang web của mình. Phần thân bắt đầu bằng thẻ và kết thúc với thẻ .

Bài 2: Các thẻ HTML thông dụng.

Thẻ mở đầu của trang HTML - none html: Thẻ chứa các thẻ trong phần đầu của trang HTML - none
head: Tiêu đề trang web - none
title: Mô tả tổng quát về nội dung trang web - none
meta: Dùng để nhúng một tập tin nào đó vào trang web - none
script: Dùng để nhúng các tập tin javascript - none
link: Dùng để bao bọc một nội dung về css - none
body: Thẻ chứa nội dung chính của website - block level
h1: Thẻ để hiện tiêu đề của một vấn đề nào đó - block level
div: Thẻ này thường dùng để chứa nội dung - block level
span: Thẻ chứa nội dung - inline
Thẻ chứa nội dung (đoạn văn) - block level
p: Thẻ canh giữa các đối tượng nằm bên trong - block level
a: Thẻ tạo link - inline
br: Thẻ xuống hàng - block level
